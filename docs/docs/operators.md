Senegal supports the operators shown below:

| Description | Operators |
| --- | --- |
| Unary Postfix | var++ var-- () [] . |
| Unary Prefix | -var !var ~var |
| Multiplicative | * / % ** |
| Additive | + - |
| Bitwise | & ^ \| << >> |
| Comparison | == != > >= < <= |
| Logical | && \|\| |
| Ternary | condition ? expr : expr  |
| Assignment | = += -= *= /= |
| Pipeline | <\| |
